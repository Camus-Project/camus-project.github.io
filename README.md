# Site – The Camus Project

Source of official [Camus Project website](https://camus-project.github.io/).

The Camus Project is an open initiative building a unified method, specification language, tooling, and programming language for software generated by AI but still understood and assumed by humans. The project develops its methods and tooling by applying them to itself — a self-hosted approach to method engineering.
